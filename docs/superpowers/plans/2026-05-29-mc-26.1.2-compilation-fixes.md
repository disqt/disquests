> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Fix all 33+ compilation errors in the MC 26.1.2 migration PR so `./gradlew build` passes and CI goes green.

**Architecture:** Systematic rename fixes organized by error category. All mappings verified against the decompiled MC 26.1.2 jar (`minecraft-merged-deobf-26.1.2.jar`), Fabric API 0.150.0+26.1.2, and owo-lib 0.13.0+26.1.
