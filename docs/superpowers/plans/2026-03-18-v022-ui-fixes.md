- [ ] **Step 2: Override mouseClicked to intercept help button clicks**

```java
@Override
public boolean mouseClicked(double mouseX, double mouseY, int button) {
    if (editing && helpButton != null && button == 0) {
        if (mouseX >= helpButton.getX() && mouseX < helpButton.getX() + helpButton.getWidth()
                && mouseY >= helpButton.getY() && mouseY < helpButton.getY() + helpButton.getHeight()) {
            helpButton.onPress();
            return true;
        }
    }
    return super.mouseClicked(mouseX, mouseY, button);
```
